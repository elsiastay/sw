Introducción ASIDE

la etiqueta ASIDE en Html se utiliza para definir el contenido que esta indirectamente relacionado con el contenido principal de una pag web. Este contenido puede ser informacion adicional.

Estructura HTML

Un documento HTML consta de elementos HTML anidados. La estructura básica incluye un elemento <!DOCTYPE html>, <html>, <head> y <body>.

Etiquetas HTML

Las etiquetas en HTML son los elementos fundamentales que estructuran y presentan el contenido en una página web. Estas etiquetas se utilizan para definir y delimitar diferentes tipos de contenido y se representan mediante pares de nombres de etiquetas encerrados entre corchetes angulares
